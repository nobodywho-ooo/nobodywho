nobodywho-wasm — chat in the browser (main-thread demo)

Minimal demo that runs Chat.ask on the main thread and drains TokenStream.nextToken() into the page. Pick a chat-style GGUF (e.g. Qwen2.5-0.5B-Instruct or Qwen3-0.6B from ~/Downloads/).

Note: tokens do NOT appear incrementally here. The Rust inference loop holds the JS thread until generation completes, so the whole response lands at once. For real-time streaming run the wasm inside a Web Worker and use askStreaming — see browser-chat-worker.html.
